QuerrySpray
Kanta Privé Publique

Software écrit en Go, prévu pour consommer de manière correcte et efficace des API défectueuses et/ou instables. J'ai développé cette première version entièrement.
J'ai aussi travaillé sur une variante packagé serverless, exploité en PHP directement via le shell_exec. J'ai maintenu le projet sur plusieurs versions, utilisées en production.
Par la suite j'ai migré le projet sur Rust pour alléger les problématiques de cross-compilation et de dépendances grace à Cargo (dev sur mac et prod sur linux).

Throttle Responder
Kanta Privé Publique

Software très simple écrit en python, prévu avec quelques autres utilitaires pour tester l'efficacité de QuerrySpray. J'ai développé le projet entièrement.
Le projet a connu de nombreuses versions. Il consiste essentiellement en un serveur flask qui permet de mesurer les temps de réponse des requêtes envoyées par QuerrySpray.
